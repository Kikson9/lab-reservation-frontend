# LabReserve - Lab Reservation System

A full-stack university lab reservation system built with React and Django.

## Project Overview

LabReserve allows university administrators to manage laboratory rooms and student reservations, and allows students to browse available labs and book seats.

## Tech Stack

**Frontend**

- React (Vite)
- Tailwind CSS
- React Router
- Axios (Phase 3 — API integration)

**Backend**

- Django
- Django REST Framework
- MySQL

## Features

**Admin**

- Dashboard with live stats and lab availability overview
- Lab management — add, edit, delete lab rooms
- User management — add, edit, delete users
- Reservation management — view, filter, cancel reservations

**Student**

- Personal dashboard with upcoming reservations
- Browse available labs with seat availability
- Book a seat with date and time selection
- View and cancel personal reservations

## Project Status

- Phase 1 — Project setup and structure
- Phase 2 — Complete UI with static data
- Phase 3 — API integration with Django backend

## Getting Started

### Prerequisites

- Node.js v18+
- npm

### Installation

```bash
git clone https://github.com/kikson9/lab-reservation-frontend.git
cd lab-reservation-frontend
npm install
npm run dev
```
